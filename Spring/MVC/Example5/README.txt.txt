В данном примере создаем форму с вводом имени. В данном примере используем Model для подмены имени и вывода уже подмененного имени.
Чтение происходит не с помощью параметра HttpServletRequest а с помощью аннотации @RequestParam
В данном примере аннотацию RequestMapping мы назначаем для класса контроллера
