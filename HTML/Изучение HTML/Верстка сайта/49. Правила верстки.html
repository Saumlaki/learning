Контакты, о нас прочее

Внутри тега header находится шапка сайта

Внутри тега nav находится навигация сайта

Внутри тега main находится навигация основная информация. Используется один раз на странице и на сайте. Блок уникальный и нигде не повторяется. Типа приветствия

Внутри тега section находится какой то сгрупированный блок инормации подходящий только для этого сайта - например на сайте по кулинарии - рецепт

Внутри тега article находится какой то сгрупированный блок инормации подходящий для любого сайта, например новости

Внутри тега aside находится любая информация(виджет соц сета, прикрепленный файл, голосование, подписка)

Внутри тега footer стаим подвал
